# 20天20个小程序
## intro
根据实战20个小程序项目来全面的介绍微信小程序，覆盖多个api接口和业务场景
由浅入深，从纯前端的展示项目 到 物流登录支付等各种api的商业项目都会进行讲解。
* 附加点：
小程序背景介绍，基础原理解析， 开发工具介绍， 测试发布部署流程介绍，
